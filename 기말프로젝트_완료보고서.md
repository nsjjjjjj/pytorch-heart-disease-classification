## 1. 프로젝트 개요

본 프로젝트는 실제 심장 질환 데이터를 이용하여 심장 질환 여부를 예측하는 이진 분류 모델을 구현한 것이다. 수업에서 배운 데이터 전처리, 정규화, 텐서 변환, DNN 구성, 손실 함수, 최적화, 학습 전후 평가 흐름을 하나의 실행 가능한 프로그램으로 연결하는 것을 목표로 하였다.

## 2. 데이터셋

- 데이터 출처: Kaggle Heart Disease CSV / UCI Cleveland Heart Disease
- 데이터 크기: 303행 x 14열
- 입력 특징 수: 13개
- 정답 라벨: target, 0=정상, 1=심장 질환 있음
- target 분포: 정상 164명, 심장 질환 있음 139명
- 데이터 분리: train/valid/test = 212/45/46

UCI 원자료의 target 값은 0, 1, 2, 3, 4 형태였으나, 프로젝트의 이진 분류 목표에 맞추어 0은 정상, 1 이상은 심장 질환 있음으로 변환하였다.

## 3. 데이터 전처리

Pandas를 이용하여 CSV 파일을 불러오고, 결측치 확인, 숫자형 변환, 입력 특징과 정답 라벨 분리, 표준화를 수행하였다. 원자료 기준 ca 열에 4개, thal 열에 2개의 결측치가 있었고, 중앙값으로 채워 303행을 유지하였다.

```python
df = pd.read_csv(HEART_CSV)
mean = train_df.iloc[:, :-1].mean()
std = train_df.iloc[:, :-1].std().replace(0, 1)
normalized_train = (train_df.iloc[:, :-1] - mean) / std
normalized_valid = (valid_df.iloc[:, :-1] - mean) / std
normalized_test = (test_df.iloc[:, :-1] - mean) / std
```

## 4. 모델 구조

모델은 `13 -> 32 -> Dropout(0.25) -> 16 -> 1 -> Sigmoid` 구조로 구성하였다. 입력층은 13개의 특징을 받고, 출력층은 심장 질환 가능성을 0과 1 사이 확률로 출력한다.

```python
self.layers = nn.Sequential(
    nn.Linear(13, 32),
    nn.ReLU(),
    nn.Dropout(0.25),
    nn.Linear(32, 16),
    nn.ReLU(),
    nn.Linear(16, 1),
    nn.Sigmoid(),
)
```

## 5. 학습 방법

손실 함수는 `nn.BCELoss`, 최적화 알고리즘은 `Adam(lr=0.001)`을 사용하였다. 최대 epoch는 300으로 설정했지만, 검증 손실이 45번 연속 개선되지 않으면 멈추는 Early Stopping을 적용하여 실제 학습은 141 epoch에서 종료되었다.

## 6. 실험 결과

| 평가 항목 | 학습 전 | 학습 후 |
|---|---:|---:|
| Accuracy | 47.83% | 80.43% |
| Recall | 42.86% | 76.19% |
| Loss | 0.7000 | 0.4386 |
| F1-score | 0.4286 | 0.7805 |

학습 후 혼동행렬은 TN=21, FP=4, FN=5, TP=16이다. 의료 데이터에서는 실제 질환자를 정상으로 예측하는 FN이 중요하므로, 단순 정확도뿐 아니라 재현율도 함께 확인하였다.

## 8. 한계점 및 개선 방향

- 데이터 수가 303개로 많지 않아 실제 의료 진단에 바로 적용하기에는 부족하다.
- 한 번의 train/valid/test 분리에 의존하므로 교차 검증을 사용하면 결과 안정성을 더 높일 수 있다.
- 의료 상황에서는 질환자를 놓치는 FN을 줄이는 방향으로 재현율 개선이 필요하다.
- DNN 외에도 로지스틱 회귀, 랜덤포레스트 등과 비교하면 모델 선택의 타당성을 더 강화할 수 있다.

## 9. 결론

이번 프로젝트에서는 수업에서 배운 인공지능 모델 학습 과정을 실제 의료 CSV 데이터에 적용하였다. 학습 전 정확도는 47.83%, 학습 후 정확도는 80.43%로 상승하였다. 이는 계획서에서 설정한 테스트 정확도 80~85% 목표 범위에 도달한 결과이다. 학습 후 재현율은 76.19%로 나타났으며, 이를 통해 전처리와 DNN 학습을 거치면 초기 무작위 모델보다 의미 있는 예측 성능 향상이 가능함을 확인하였다.
